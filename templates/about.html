About VPC Flow Log Analyzer

What is VPC Flow Log Analyzer?

A powerful tool designed to help you understand and analyze AWS VPC Flow Logs. It provides insights into network traffic, security group suggestions, and visualizations.

Key Features

  • Detailed traffic summary
  • Per-ENI security group rule suggestions
  • Interactive traffic visualization
  • Protocol and connection insights

How It Works

  1. Export VPC Flow Logs from AWS Console
  2. Upload the log file to the analyzer
  3. Get comprehensive network traffic analysis

Technologies Used

Backend

Python, Flask, Pandas

Frontend

Bootstrap, Plotly

Visualization

Interactive Charts
